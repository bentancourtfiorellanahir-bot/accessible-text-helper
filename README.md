# Accessible Text Helper

Proyecto en Python orientado a accesibilidad.

## ¿De qué trata?

Este proyecto busca crear una herramienta simple que ayude a personas a interactuar mejor con el texto, especialmente en contextos de educación especial.

Incluye dos funciones principales:

- Convertir texto a voz (text-to-speech)
- Simplificar textos para que sean más fáciles de entender

La idea es combinar ambas opciones para facilitar la comprensión y el acceso a la información.

## ¿Para quién está pensado?

Está pensado principalmente para:

- Personas con discapacidad visual
- Personas con dificultades de comprensión lectora
- Contextos educativos inclusivos

## Funcionalidades

- Leer texto en voz alta
- Simplificar textos mediante reemplazo de palabras complejas
- Opción de simplificar y luego escuchar el texto
- Interfaz en consola con instrucciones claras

## Tecnologías utilizadas

- Python
- pyttsx3
